Обучить модель классификации изображение на 2 класса.

Исходные данные и валидация решения на kaggle в рамках контеста Cats vs Dogs. Шаблон ipython-ноутбука для решения можно скачать по ссылке.

Решения необходимо прислать в виде ipython-ноутбука с указанием значения метрики на Leaderboard. Задание засчитывается при значениях метрики Log Loss меньше 0.3. В случае нехватки ресурсов и сложностей с получением лосса 0.3 у модели VGG16 в этом задании можно использовать модель MobileNetV2.
